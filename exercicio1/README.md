# Exercício 01 - Calculadora em C

Este exercício tem como objetivo praticar os conceitos básicos da linguagem C, utilizando funções e um menu interativo.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão (com tratamento de divisão por zero)

## Conceitos praticados
- Funções
- Estrutura `switch`
- Laço de repetição `do while`
- Entrada e saída de dados

## Como executar
O programa pode ser compilado e executado no **DevC++**:

1. Abra o arquivo `main.c` no DevC++
2. Compile e execute usando **F11**
